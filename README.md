# 📺 电视投屏助手 Cast Helper

把不带「投屏」按钮的网站视频,投到支持 Chromecast 的电视上。提供两种形态:一个纯静态网页,一个安卓 App。无后端、不收集任何数据。

> ⚠️ 仅限投放**你自己有权观看**的内容(自己的会员、自己买的课等);不支持、也不会绕过 DRM(Netflix / Disney+ 等)。

- **在线网页**:<https://yingwang.github.io/cast-helper/>(需要电脑或安卓上的 Chrome / Edge)
- **安卓 App**(推荐 Boox / 安卓手机):见下方[安卓 App](#-安卓-app推荐给-boox--安卓手机)一节。

顶部都内置了「常用视频网站」快捷入口(哔哩哔哩、芒果TV、优酷、腾讯视频、爱奇艺、西瓜视频、YouTube、爱壹帆、欧乐影院),点一下直达,不用手打网址。

## 🖥️ 网页版怎么用

打开在线网页,任选一种方法投屏。

### 方法一 · 最简单:Chrome「投放标签页」

不用本工具的任何功能,Chrome 自带,几乎所有网站都能用:

1. 电脑 Chrome 打开视频网站,登录并播放;
2. 右上角 **⋮ → 投放 → 选电视 → 「来源」选「投放标签页」**;
3. 视频**全屏**播放,Chrome 会自动优化画质。

> 这是屏幕镜像,一般 720p–1080p,追剧够用。安卓手机可用下拉快捷设置里的「屏幕投射 / 无线投屏」;iPhone / iPad 浏览器不支持,请用电脑或安卓。

### 方法二 · 画质最好:抓直链,电视自己拉流

电视直接从源站拉流,原生清晰度。**配置一次,之后一键投**,不用复制粘贴:

1. 打开在线网页,把里面的「🎯 抓视频直链」按钮拖到浏览器书签栏(只需一次);
2. 到视频网站让视频**先播放几秒**,点一下这个书签;
3. 它会**自动打开本工具并把直链填好**,你直接点「📺 投到电视」——工具页自带播放 / 进度 / 音量遥控条。

> 有些会员站的直链带防盗链或不允许跨域,电视会拉不动,退回方法一即可。`blob:` / DRM 内容抓不到,也用方法一。

## 📱 安卓 App(推荐给 Boox / 安卓手机)

App 内自带浏览器,自动嗅探视频、一键投屏,比网页版更省事,不用弄书签、不用复制链接。地址栏下方就是常用网站快捷按钮,点一下直达(Boox 上尤其省事)。

用法:**选 / 输网站 → 登录并播放 → 出现提示后点「📺 投屏到电视」**。第一次会弹出「选择电视」让你选一次,选好后自动开始投;之后点一下按钮就直接投,不用再另点右上角的投屏图标。

投屏后下方出现遥控条:顶部显示电视名和播放 / 缓冲状态,按钮有播放 / 暂停、-10s / +30s、⏭ 下一集(手动触发网站连播,不用回网页找选集)、⏹ 停止,还有进度条、倍速(0.5–2x)、音量与静音(调的是 Chromecast 输出,和电视遥控器音量叠加生效)。电视上会显示当前视频的标题(取自网页标题,一般就是剧名和集数);按「⏹ 停止」或断开电视时,会把电视看到的进度写回网页播放器,回手机上能接着看。投屏默认投出网页当前选中的固定码率线路,和网页里放的清晰度一致、最稳(早期试过投 m3u8 总表让电视自适应,但不少站总表拉不动或频繁缓冲,故改回固定线路)。另有「自动连播(实验)」开关:一集放完自动触发网站自带的连播并接投下一集——仅在支持连播的网站上有效,接不上会提示回 App 手动点。投屏期间通知栏会挂一个播放通知,让 App 在息屏 / 后台时也能继续工作(个别设备杀后台激进的话,自动连播仍可能被打断)。

和网页版方法二一样,少数站的直链带防盗链或要求跨域头,电视会拉不动(App 会提示「电视拉流失败」),这类站请改用电脑 Chrome 的「投放标签页」(方法一)。走加密 / 内存流(blob)的站(如部分哔哩哔哩视频)嗅探不到直链,App 会在状态栏提示,同样请用方法一。

- **下载安装包**:<https://github.com/yingwang/cast-helper/releases/download/app-latest/cast-helper.apk>
- 安装:Boox 上用浏览器打开上面的链接下载,点开安装;若提示,请在「设置 → 安全」里允许该浏览器「安装未知来源应用」。
- 升级:从 1.0(旧 debug 签名)升到 1.1+ 需先卸载旧版一次(签名已固定);此后的版本可直接覆盖安装。
- 前提:设备装有 Google Play 服务,且与电视连同一 Wi-Fi。
- ⚠️ 仅安卓可用;**iPhone / iPad 无法安装非 App Store 应用**,请用在线网页,或 Web Video Caster 之类现成 App。

> 安装包由 GitHub Actions 自动编译发布([Build Android APK](../../actions/workflows/android.yml)),源码见 [`android/`](android/);本地构建:`cd android && ./gradlew assembleRelease`。
> 签名密钥就放在仓库里(`android/app/release.keystore`,口令见 `build.gradle`):它只用来保证「同一来源可覆盖升级」,不承担保密;介意冒充风险可改为从 GitHub Secrets 注入。

## 限制

- 仅供投放**你自己有权观看**的内容;不支持、也不会绕过 DRM(Netflix / Disney+ 等不可用);
- iPhone / iPad 的浏览器不支持 Chromecast 投屏,请用电脑或安卓设备;
- 你的设备需要和电视连在同一网络。

## 本地运行

下载 `index.html`,在其所在目录运行:

```bash
python3 -m http.server 8000
```

然后浏览器打开 <http://localhost:8000>(投屏组件要求页面通过 `https://` 或 `localhost` 打开,双击文件的 `file://` 方式不行)。
