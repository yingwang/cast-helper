方法一(最简单,先试这个):Chrome 直接"投放标签页"

不需要本页面的任何功能,Chrome 自带,几乎所有网站都能用:

  1. 电脑上用 Chrome 打开你的会员网站,登录并播放视频;
  2. 点右上角 ⋮ 菜单 → 投放、保存和分享 → 投放…(旧版 Chrome 菜单里直接叫「投放…」);
  3. 选择你的电视,「来源」选 投放标签页;
  4. 把视频全屏播放——全屏后 Chrome 会自动优化画质与流畅度。

安卓手机:下拉快捷设置里的「屏幕投射 / 无线投屏 / Smart View」,或用 Google Home App 镜像屏幕。 iPhone / iPad:浏览器无法向 Chromecast 投屏,请改用电脑或安卓设备。

方法二(画质最好):抓视频直链,让电视自己播放

电视直接从源站拉流,不经过你的电脑镜像,清晰度和流畅度都是原生的。步骤:

  1. 把这个按钮拖到 Chrome 的书签栏(按 Ctrl/⌘ + Shift + B 显示书签栏)——只需一次:

    🎯 抓视频直链
  2. 打开你的会员网站,让视频先播放几秒;

  4. 在弹出的工具页点「📺 投到电视」即可。

常见问题

投屏组件加载不出来 / 找不到电视?
  • 必须用 Chrome 或 Edge(电脑或安卓);iOS 上任何浏览器都不行。
  • 你的设备要和电视连同一个 Wi-Fi。既然你能用 YouTube 投屏,说明网络是通的,这条一般没问题。
  • 本页需要通过 https:// 或 http://localhost 打开。如果你是双击本地文件打开的(地址栏是 file://),投屏按钮可能失效——在文件所在目录运行 python3 -m http.server 8000,再访问 http://localhost:8000。
直链投过去电视黑屏 / 转圈 / 报错?
  • 很多会员站的视频地址带防盗链(校验 Cookie、Referer 或 IP),电视自己去拉流会被拒绝。这种情况请退回方法一「投放标签页」,它是镜像你的浏览器画面,不受防盗链影响。
  • HLS(.m3u8)直链还要求源站允许跨域(CORS),不允许的也只能用方法一。
  • 先点「用测试视频试一下」:测试视频能投说明链路正常,问题出在那个网站的直链上。
Netflix / Disney+ 这类网站能用吗?

不能。这类内容有 DRM 加密,抓不到直链,镜像时也会黑屏——这是版权保护机制,本工具不会也无法绕过。这类平台请用电视上自带的官方 App 登录观看。

书签点了说"没找到视频直链"?
  • 先让视频真正播放几秒再点;
  • 有些网站的播放器在 iframe 里:在播放器上右键 →「在新标签页中打开框架」,在新页面里再点书签;
  • 地址是 blob: 开头或 DRM 内容的,抓不到,用方法一。
两种方法画质差多少?

方法二是电视原生拉流,能到源站提供的最高清晰度;方法一是屏幕镜像,通常约 720p~1080p、30 帧,追剧完全够用,全屏播放时 Chrome 会自动走优化通道。
